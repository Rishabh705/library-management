Library API Documentation

Authentication Routes

POST /login

Description: Authenticates a user and returns a token.

Request Body:

{ "username": "string", "password": "string" }

Response:

200 OK: Returns an authentication token.

{ "token": "string" }

400 Bad Request: Invalid credentials.

POST /register

Description: Registers a new user.

Request Body:

{ "username": "string", "password": "string" }

Response:

201 Created: Successfully created a new user.

400 Bad Request: Invalid input.

Book Routes

GET /books

Description: Retrieve a list of all books in the library, with optional search and pagination.

Query Parameters:

title: "string" (optional) author: "string" (optional) page: 1 (optional) per_page: 5 (optional)

Response:

200 OK: Returns a list of books.

[ { "id": "string", "title": "string", "author": "string", "year": "string" }, ... ]

POST /books

Description: Add a new book to the library.

Request Body:

{ "title": "string", "author": "string", "year": "string" }

Response:

201 Created: Successfully created a new book.

401 Unauthorized: Token required or invalid token.

GET /books/{book_id}

Description: Retrieve details of a specific book by its ID.

Response:

200 OK: Returns book details.

404 Not Found: Book with the specified ID not found.

PUT /books/{book_id}

Description: Update details of an existing book.

Request Body:

{ "title": "string", "author": "string", "year": "string" }

Response:

200 OK: Successfully updated the book.

401 Unauthorized: Token required or invalid token.

DELETE /books/{book_id}

Description: Delete a specific book by its ID.

Response:

200 OK: Successfully deleted the book.

401 Unauthorized: Token required or invalid token.
